Programa UpAcademy

Receita-me foi o nosso primeiro projecto de grupo no âmbito do curso JAVA Starter da HRB UpAcademy. A aplicação web foi desenvolvida a partir de JavaServer Faces (JSF) com componentes de PrimeFaces, Bootstrap, SQL, e Javascript. O projecto teve a duração de 3 semanas e foi planeado com base no método SCRUM-AGILE.

Projecto Receita-me

Receita-me é uma interface de busca de receitas com base nos ingredientes introduzidos. O utilizador tem a possibilidade de introduzir os ingredientes disponíveis em casa e pesquisar por receitas relevantes que contêm o maior número de ingredientes em comum. Adicionalmente, o utilizador pode efectuar uma pesquisa avançada com base no valor nutricional de cada receita.
Caso o utilizador pretenda submeter uma nova receita, a interface possui um sistema de submissão que passa pela aprovação do administrador antes de ser introduzida na base de dados.
