A Equipa

Programa UpAcademy

Receita-me foi o nosso primeiro projecto de grupo no âmbito do curso JAVA Starter da academia UpAcademy. A aplicação web foi desenvolvida a partir de JavaServer Faces (JSF) com componentes de PrimeFaces, Bootstrap, SQL, e Javascript. O projecto teve a duração de 3 semanas e foi organizado com base no método SCRUM-AGILE.

Projecto Receita-me

Receita-me é uma aplicação web de busca de receitas desenhada a pensar na redução do desperdício alimentar. A interface convida os usuários a introduzirem os alimentos que têm em casa, e a pesquisarem por receitas que englobem o maior número de ingredientes em comum. O website também permite ordenar as receitas procuradas com base no seu valor nutricional, e a submissão de novas receitas mediante a aprovação de um moderador.
